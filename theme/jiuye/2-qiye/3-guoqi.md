这一章节，不讨论具体是国企还是央企，后文聊国企基本上就指代国企和央企两类，指国资企业。

公务员是行政编，事业单位是事业编，而国企，除了部分领导之外，是没有编制的。

# 为什么去国企

同学毕业之后的发展，最重要的东西不是努力，而是做选择。

刚出社会的时候，会觉得自己三头六臂，一定能在社会、在工作上创出一片天，只要自己努力，只要自己足够卷，只要自己足够认真，就一定能成。

当然了，我们都赞同这种想法，精神上赞同，最终结果如何，那是看你自己的命运，我个人从来都不推崇：从技术的角度入手，努力学习、努力工作，然后走上专家的道路。

扯远了。

不管是谁，只要是选择工作，就必须做一些取舍：你如何衡量家庭、工作、薪资这三件事。

当你在私企工作好几年，特别是私企的工资很高的情况下，你想跳槽去国企工作，大部分的情况就是你的薪资会有很大的缩水，但是你仍然是可以接受的，是因为你觉得自己需要更多的个人时间去陪伴家庭和给予自己。

绝大部分国企的现状就是：信息化水平比较落后，技术研发体系比较落后（针对计算机互联网技术类说），而整个工作的方式信息化程度不高，手续报销审批也是一层又一层，这很正常。

而如果你去私企，甚至去一些一线的企业，你可能能用上最好的工具、最好的办公电脑、最好的办公环境，然而你需要牺牲的可能就是你自己的时间！由于我们仍然并且长期处于发展中国家的状态，所以加班问题可能你就不要想着私企能够解决了（工资高的私企）（好吧，有的工资低的私企加班也严重）。

但是，既然希望毕业后就回老家工作挣钱买房定居，或者出来在外打拼也没有一个认为适合自己的归宿，为何不直接选择离家近的国企呢？钱多事少离家近，有两个就可以说是完美的工作，有一个那就是很合适的工作了。

# 真实的国企可能不那么理想

在互联网上，大家认为的国企，一般都是在市场上有垄断型地位的，而不是市场化竞争的国企，就像电网运营商银行这样的，一进去就是坐办公室的轻松闲职。（当然了，这只是想象中的理想状态）

**网上的国企可能是这样的：**

- 工资七八千随随便便，加上各种福利月入万元轻轻松松；
- 8小时工作制不加班；
- 不会辞退员工，随便混日子。

**真实的国企：**

- 工资七八千的都是高端岗位，二线城市月入3000多的国企多的是，三四五六七八线城市的国企到手工资2000多点的多如牛毛；
- 大部分确实能够达到8小时工作制，但是也有很多工作岗位比私企劳动强度还大，还有的岗位你也坐不了办公室，在工地，在外面到处跑；
- 基本都是校招进来的，或者是领导的子女走后门进来的，还有就是工厂子弟，家里有人照顾着进来的，而社招基本上不招人，即使招人也是挖高端岗位的领导过来。

国企的特点就是，收入可能比较一般（不用举那些特例了），但胜在稳定性比较强，对于希望能够稳定的人来说，国企有着很强的吸引力。

尤其是对于一些并不是特别高学历的同学，大学毕业就去离家比较近的国企，是很不错的选择，工资平均上来说也会比私企高，工作时间一般比较正常，补贴多，五险一金福利待遇会比较完善。

国企为啥要卡学历吗，因为普遍状况下，凭本事进来的学东西快适应力强，做事能力大，就得多做事。因为不管怎样总有关系户进来，像我这样做事的还好，也有些真的只是进来找个班上混日子的，也有凭本事进来有些失望不想做事的也在混日子的，都不做事总得招几个能干的做事，有人在混日子，就有人在负重前行。

确实一般国企很少开除人，因为你的领导可能不知道你后面到底站着谁，开除你得罪人谁都不乐意，真的可能你今天开除了谁，明天就被穿小鞋，这都还是好的，也有那种今天开除，不出半年这个领导就进去的，不要说谁干净，当然这种就是被开除的人手里有领导的把柄，然后后面也有人，气不过就弄你。

所以国企相对于私企，还是有一定不同的：**国企有门槛，国企稳定上限低，私企不稳定但上限高，机会也多，但是比较冒险主义。**

# 什么叫假国企

国企转私企这块现在国家正在大力推动混合所有制改革，就是把国有企业吸引民营资本、集体、外资加入，成为混合股权的企业。这分两种情况：

1、国企通过混改吸引了私企等加入，变成国有控股公司。这种公司还属于国有企业序列，是国企控股，民营资本在里面占股，然后市场化转型。这类企业本质上还是国企，职工身份还是没有变。

2、国企通过混改出让了大部分股权，成为国有参股公司。这种公司基本上等同于私企了。这类企业完全按市场化运行，但因为是国有参股，所有国有影响还是有的，并且可以整合借助国有的一些资源等，前景还不错。

所以，传统意义上的国企，其实就是国资委全部控股的那些垄断型国企，每个地区都会有其分部。

# 国企也会是围城

国企内部的氛围，并没有外面想的那么好混。除去一些真的非常传统的、或者人特别充足的国企，每天的工作量，也是比较饱和的。至少我所在部门当初的工作，十有八九都是因为上层领导为了政绩要求，拍脑袋想出来的课题，底下的人就是围着这些大概率落不了地的事情进行调研、规划、不断试错。每天有大量的数据和PPT需要处理。

另一方面，国企对于为人处事方面的要求，是非常高的。这个属于隐性关卡，因为你永远不知道和你对话的上级、平级、下级，他的七大姑的表妹的老丈人的外甥是集团里的哪个领导。因此，怎么站队、如何讲话、怎么与别人在项目中共事，都是非常讲究政治正确的。这方面，很多民企外企相对来说会简单很多，毕竟关系户太多的结果，又没有垄断能力，那在市场化竞争就是一个失败。

对于有能力但没关系的人来说，国企能提供的仅仅是稳定的饭碗，薪资待遇和晋升途径大多数是没法摆在市面上竞争的，有时候可能是好几倍的差别。要想在国企致富，除非未来有把握能晋升或熬成管理层，否则基本上没戏，一辈子可能还没别人在外头打工到40岁、提前退休挣得多。

所以这就是个围城，最终取决于自己想要什么吧。

**没能力没关系想进去混，那还是可以考虑的，就怕你心比天高混不住，拿着永远稳定的工资被其他同龄人超越的时候，却又觉得一开始不该混那种。**

国有企业的工资水平，是受到国资委严格管控的，而且国资委对国企有工资总额控制。所以，国有企业在一个水平之后，就再也升不了了。这是一个很大的问题。对于能力特别强的人来说，很吃亏的。此外，国有企业仍然存在较大程度的大锅饭。无论怎么改革，大锅饭还是存在的。还有，最关键的一个问题，就是流动性，很多国企缺乏流动性。这是很要命的。比如某个领导看你不顺眼，你只能忍着等他调走。而你很难跳槽。

# 为什么大家都说国企好

其实只是打工人表达不满。

就好比我们一些人总是提到苏联，就好比韶山12.26的游客越来越多。其实这是群体的无意识自组织，是对切肤之痛的抗争。私企的兴起时间太短了，中国这十几年突变式发展，给不少人带来了迷幻感：上一辈还是四点下班去幼儿园接的我，现在我怎么连十一点下班都做不到了？其实现在这十年国企早就风头不再，要想发财莫入此门，有发财想法的纪委早就逮走了。福利也越来越少。但是现在就是批判体制拿国企出来当靶子，争取权益的的时候拿国企出来当典型。

我国现有体制下本来就应该理直气壮地实行双轨制。

现在就是国企遮遮掩掩的混改、改制，钱不多发一分人砍掉三分之一，叫唤着向私企效率学习。私企恬不知耻提倡奉献、情怀，无偿加班无限延长。为什么两边不多思考一下，为什么都变成了四不像？

公有制就是公有制，道路自信在哪里？

一个学生进入社会后可以根据自身条件和需求，或者选择去私企激情打工，或者选择去国企大机器里做一颗螺丝钉。想要朴实平安的往左走，想要赚钱跃升的往右走。

只不过现在往右走的朋友们发现，钱也没挣着，鸡毛也没跃升，身体还垮了。不由得喊一句：

“国企真好。”

**总结：对于打工人来说，不是国企太好，而是私企太烂了，然而你说对于打工人来说私企烂是私企的问题吗？恐怕还真的不是。。。外企也是私企呢。。。外企来自资本主义社会呢。。。**

# 国企的两面

**好的一面：**

- 上班时间稳定，每天只工作7小时，周末稳定双休。
- 五险一金足额交，补充医疗这块做的蛮好，医疗费、药品费都可以报销，仅药费来说一年可以报销到8000+，公积金有的单位会把缴存基数提的特别高。
- 对于女性比较友好，不会因为你生孩子辞退你，领导也不会劝你晚点生孩子。
- 公司有食堂，早中饭在公司吃，每个月公司充的饭卡钱都用不完。
- 人到中年，找工作我就看2点：一是稳定，绝不能在我40～45时候随随便便踢社会上自生自灭去，二是性价比要高，工资可以低，事儿不能多。
- 国企效率低，体制僵化，一个月只能干私企一周的工作。（是的没错，这是优点，你要是觉得这是缺点说明你太年轻了）

**不好的一面：**

- 升职很难，特别熬人，大部分人干一辈子也只是基层工人，一眼就能看到三十年以后的生活，拿着固定的薪水，过着一成不变的生活，想想都绝望。
- 升职是个玄学，不在于你能力有多强，就看运势来没来，每个升迁的人背后或多或少都有些让人会心一笑的故事。
- 可能是一个压抑、充满抱怨、看不到任何希望的地方。薪资不高、离家远、消耗着……
- 论资排辈，不看你的工作量/工作难易程度大小，工资都是按年限来发放，同级别的老员工每天在那喝茶、刷手机、聊天侃地过一天，一个月工资或者奖金照样能比勤勤恳恳的你多0.5-1倍。
- 官僚主义和形式主义很严重，有关系的人和会溜须拍马的人往往会比老实做事的人混的好。（当然了私企也是一样的，只不过关系户没那么多罢了）
- 有关系才轻松。无关系累成狗。或者关系不硬也累成狗。国企就是看关系。而且无论你累不累工资都差不太多的。因为一个国企大家都不干活不就废了。所以养一个人。就得让另一个人干两份工。
- 压力层层传递，最大压力集合体就在科级小干部，他们真是累死累活，基层员工只需报数据给他们汇总整理，并且基层员工可以对结果不用负责任。
- 很多人都不满现状，可是又很少有人敢辞职。因为在小城市，就业环境很差，放弃一份相对稳定的工作，到外面打零工或许还不如在国企。
- 安稳过头了，没有裁员风险，但是以上种种也会让你升不起上进的念头。最大的收获就是：**只要你不上进，那么你就会活得很开心。**
